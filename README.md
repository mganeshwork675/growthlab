# GrowthLab — Digital Marketing Agency Website

A modern, fully responsive single-page agency website built with HTML, CSS, and vanilla JavaScript.

## Features
- Hero section with key performance stats
- Services showcase (Google Ads, Meta Ads, Brand Promotion, Landing Pages, Shopify, WooCommerce)
- How It Works — 4-step process breakdown
- Results & metrics section
- Tiered pricing packages
- Client testimonials
- Lead generation contact form
- Scroll-reveal animations & marquee ticker
- Mobile responsive layout

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Syne + DM Sans)
- No frameworks, no dependencies

## Usage
Just open `https://growthlab.work-ganeshmohapatra675.workers.dev/` in any browser — no build step required.

## Deployment
Hosted on [Cloudflare/ GitHub Pages] — deploy by integrating the HTML file.
